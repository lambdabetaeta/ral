# Exit code hints for ral shell
#
# Format: <command> <status> <hint text>
#
# <command> is the basename of the executable, or * for any command.
# Lines starting with # and blank lines are ignored.
#
# Command-specific entries are checked before wildcard (*) entries.
# Signal-killed exits (status 128+N) are handled programmatically.

# grep family: status 1 = no matches (normal), 2 = usage/IO error
grep    1   no matches found
grep    2   syntax or usage error
egrep   1   no matches found
egrep   2   syntax or usage error
fgrep   1   no matches found
fgrep   2   syntax or usage error
rg      1   no matches found

# diff: status 1 = files differ (normal), 2 = trouble
diff    1   files differ
diff    2   an error occurred

# test / [ : status 1 = condition was false
test    1   condition was false
[       1   condition was false

# curl exit codes
curl    6   could not resolve host
curl    7   could not connect to host
curl    22  HTTP error (4xx or 5xx)
curl    28  operation timed out
curl    35  SSL/TLS handshake failed
curl    56  failure receiving network data

# ssh
ssh     255 connection or protocol error

# Generic codes (any command)
*       126 command found but not executable
*       127 command not found
